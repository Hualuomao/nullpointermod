一个我的世界模组
这个mod添加了一个特殊的武器，可以射出java.nullPointerException抛射物。
对实体的命中会立即杀死它们。
击中块或超时将使用 NullPointerException 使游戏崩溃。
A Minecraft mod
This mod adds a special weapon that shoots java.NullPointerException projectiles.
Hits on entities will instantly kill them.
Hits on blocks or timeout will crash the game with a NullPointerException.
